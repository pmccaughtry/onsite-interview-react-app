# Onsite Interview Coding Exercise

## Instructions

### Part 1

Design the web page shown in the following linked image: [Coding Challenge Output](./coding_challenge.jpg)

* The page should be responsive
* The shadows around the browser are not part of the design

### Part 2

1. Use the data from the following endpoint to build out the repository list - [Github Repositories with > 10000 Stars](https://api.github.com/search/repositories?q=stars:>10000&sort=stars)
1. Enable filtering on the language links to display all repositories that match the filter (All, JavaScript, Ruby, Java, CSS, Python)

## Run the Application
Run `npm start` from a terminal window to launch the application.
